### 문제 주소
[Delete Node in a Linked List](https://leetcode.com/explore/interview/card/top-interview-questions-easy/93/linked-list/553/)

### 접근 방법
`node`의 값(`val`)을 다음 `node.next`의 값으로 변경하고  
연결된 `next`를 현재 `node`의 다음 다음의 `node`로 연결

### 시간복잡도
O(1)
